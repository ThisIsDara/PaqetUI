# PaqetUI

یک رابط گرافیکی مدرن برای پروکسی شبکه‌ای paqet.

[![لایسنس](https://img.shields.io/github/license/ThisIsDara/PaqetUI)](https://github.com/ThisIsDara/PaqetUI)
[![نسخه](https://img.shields.io/github/v/release/ThisIsDara/PaqetUI)](https://github.com/ThisIsDara/PaqetUI)

## ویژگی‌ها

- تم تاریک مدرن (الهام‌گرفته از Discord/Linear)
- پشتیبانی از حالت کلاینت و سرور
- تشخیص خودکار رابط شبکه
- تنظیمات رمزنگاری KCP
- ثبت زنده فرآیند
- Import/Export فایل‌های YAML
- تنظیمات پایدار

## اسکرین‌شات‌ها

> اسکرین‌شات‌های خود را اینجا اضافه کنید

## دانلود

آخرین نسخه را از [GitHub Releases](https://github.com/ThisIsDara/PaqetUI/releases) دریافت کنید.

## نحوه استفاده

1. `PaqetUI.exe` را اجرا کنید
2. حالت کلاینت یا سرور را انتخاب کنید
3. تنظیمات شبکه را پیکربندی کنید
4. روی **START TUNNEL** کلیک کنید

## پیش‌نیازها

- ویندوز 10/11
- Npcap (برای ویندوز)
